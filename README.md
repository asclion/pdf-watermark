# pdf_watermark

## Description

Add a watermark to a PDF document.

@date     2015-03-11
@version  0.1
@author   meuniern@laposte.net

## How to run?

$ gem install bundle
$ bundle install
$ ruby pdf_watermark.rb <input-file> <output-file> <watermark-text>

## How to convert this ruby script as Windows executable?

$ gem install ocra
$ ocra pdf_watermark.rb
